## Live Data Analytics Engine: A Comprehensive Overview

**Live Data Analytics Engine** is a lightweight, modular C++ project designed for real-time data simulation, collection, and analysis. Its primary goal is to provide a robust and efficient framework for understanding and processing continuous data streams, making it ideal for applications in IoT, financial analytics, sensor data processing, and more. The modular architecture ensures easy maintainability, extensibility, and integration with other systems.

### ðŸ§© Core Components

The project is structured around three main components, each with a distinct role in the data pipeline:

1.  **StreamSimulator**:
    *   **Function**: Responsible for generating synthetic data streams that mimic real-world data sources.
    *   **Details**: Produces continuous data points, typically including a `timestamp` and a numerical `value`. This allows for testing the data processing pipeline without needing a live external data source.

2.  **DataCollector**:
    *   **Function**: Manages the ingestion and storage of data generated by the `StreamSimulator`.
    *   **Details**: Can store data temporarily in memory (RAM) for rapid access or persistently in files for longer-term retention and analysis. Ensures data integrity and prevents loss during transit.

3.  **AnalyticsCore**:
    *   **Function**: Performs preliminary and statistical analysis on the collected data.
    *   **Details**: Capable of calculating basic statistical metrics such as mean, variance, and standard deviation. Designed to be extensible, allowing for the integration of more complex analytical models, including machine learning algorithms.
